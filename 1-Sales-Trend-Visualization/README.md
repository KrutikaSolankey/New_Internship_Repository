# Sales Trend Visualization

## Project Overview

This project analyzes sales data to identify trends and patterns using Python. The data is cleaned, analyzed, and visualized using graphs to provide business insights.

## Objective

* Analyze sales performance over time.
* Identify top-selling products.
* Compare sales across different countries.
* Visualize sales trends using charts.

## Tools and Technologies

* Python
* Pandas
* Matplotlib
* Seaborn
* Google Colab

## Dataset

The dataset contains sales transaction information including:

* Order Date
* Product Line
* Sales Amount
* Country

## Steps Performed

1. Imported the dataset.
2. Cleaned missing values.
3. Converted date columns into proper format.
4. Performed sales analysis.
5. Created visualizations.
6. Generated insights from the data.

## Visualizations

* Monthly Sales Trend
* Top Selling Products
* Country-wise Sales Analysis

## Key Insights

* Highest sales month was identified from the sales trend graph.
* Top-performing product category generated maximum revenue.
* Sales contribution was compared across countries.
* Seasonal variations in sales were observed.

## Conclusion

The project successfully analyzed sales data and visualized important business trends. These insights can help organizations make better sales and marketing decisions.
