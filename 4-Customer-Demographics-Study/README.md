# Customer Demographics Study

## Objective

To analyze customer demographic data and identify patterns in customer behavior.

## Tools Used

* Python
* Pandas
* Matplotlib
* Google Colab

## Dataset

Customer Demographics Dataset

## Steps Performed

1. Imported dataset.
2. Analyzed gender distribution.
3. Compared spending patterns.
4. Created visualizations.
5. Generated insights.

## Output

* Gender Distribution Pie Chart
* Spending Comparison Bar Chart

## Conclusion

The project helps understand customer demographics and spending behavior, which can support business decision-making.
