# Personal Expense Analytics

## Objective

To analyze personal expenses and identify spending patterns using Python.

## Tools Used

* Python
* Pandas
* Matplotlib
* Google Colab

## Dataset

Personal Expense Dataset (CSV)

## Steps Performed

1. Imported expense data.
2. Cleaned the dataset.
3. Categorized expenses.
4. Calculated total spending.
5. Created visualizations.
6. Generated spending insights.

## Output

* Category-wise Expense Analysis
* Expense Distribution Pie Chart
* Spending Comparison Bar Chart

## Conclusion

The project helps understand spending habits and identify areas where expenses can be optimized.
